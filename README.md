# DjangoWebSite
- 목표 : 백엔드 프레임워크인 django를 활용하여 커뮤니티 사이트의 기본 틀인 블로그 기능 구현

# 사용방법
1. Repository clone
2. 가상환경 설치 : python -m venv myvenv
3. django 설치 확인 : pip freeze
4. 가상환경 실행 : source myvenv/script/activate
5. 서버 실행 : python manage.py runserver

# 주요 기능
- 로그인/로그아웃
- 게시글 작성
- 파일 업로드 기능
- 댓글 작성
